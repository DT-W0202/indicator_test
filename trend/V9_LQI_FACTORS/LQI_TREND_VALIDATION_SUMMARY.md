## 推荐使用方式

### 1. 崩盘预警系统 (日度)

```python
# 计算各因子百分位 (5Y滚动窗口)
vix_pctl = rolling_percentile(T7_VIX, 1260)
skew_pctl = 100 - rolling_percentile(T3_SKEW, 1260)  # 反向
vvix_pctl = rolling_percentile(T2_VVIX, 1260)
funding_pctl = 100 - rolling_percentile(T6_Funding, 1260)  # 反向

# 综合崩盘风险评分
crash_risk = 0.4 * vix_pctl + 0.3 * skew_pctl + 0.2 * vvix_pctl + 0.1 * funding_pctl

# 风险等级
if crash_risk > 80:
    level = "CRITICAL"  # 历史崩盘率 ~40%
elif crash_risk > 60:
    level = "HIGH"
elif crash_risk > 40:
    level = "ELEVATED"
else:
    level = "NORMAL"  # 历史崩盘率 <5%
```

### 2. 与 EWI 集成

```python
# EWI V4 架构建议

    final_risk = max(EWI_Strategic, 80)  # 战术层Override
```
