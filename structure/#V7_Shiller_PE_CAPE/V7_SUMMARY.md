## 关键发现

### 1. 全样本 IC 显著为负

| Horizon | IC | p-value |
|---------|-----|---------|
| 6M | -0.20 | 0.0005 |
| **12M** | **-0.31** | **<0.0001** |

**解读**: 高 CAPE → 低未来收益，这是经典的估值均值回归

### 2. 低利率环境下效果更强 (重要发现!)

**交互项回归**: R = α + β·CAPE + γ·Rate + δ·(CAPE×Rate) + ε

| 系数 | 值 | HAC t-stat | p-value |
|------|-----|------------|---------|
| const | 0.3203 | 3.40 | 0.0008 |
| **factor** | **-0.0085** | **-2.08** | **0.039** |
| condition | -0.0429 | -0.72 | 0.47 |
| interaction | 0.0009 | 0.56 | 0.57 |

**R² = 0.13**

**Regime-Split IC**:

| Regime | N | IC | p-value |
|--------|---|-----|---------|
| High Rate | 147 | -0.03 | 0.71 |
| **Low Rate** | 158 | **-0.37** | **<0.0001** |

**关键发现**:
- 交互项不显著，但 regime-split 显示明显差异
- **低利率环境下 CAPE 预测力极强 (IC=-0.37)**
- **高利率环境下 CAPE 几乎无效 (IC=-0.03)**

**解释**:
- 低利率 → 投资者追逐收益 → 估值泡沫更容易形成
- 低利率 → 未来收益压缩 → 高估值更难维持
- 高利率 → 估值自然受压 → CAPE 信号被利率因素主导

## 经济机制解释

### 为什么 CAPE 是有效的估值因子？

**核心规律**: 估值均值回归

**机制**:
1. 高 CAPE = 价格相对盈利过高
2. 长期来看，价格/盈利比率会回归均值
3. 回归方式：价格下跌 或 盈利上升
4. 高起点 → 低预期回报

### 为什么低利率时 CAPE 更有效？

**解释**:
1. 低利率 → 股票相对债券更有吸引力
2. 资金涌入 → 推高估值
3. 估值泡沫更容易在低利率环境形成
4. 泡沫破裂时 CAPE 的预测力凸显

### 为什么高利率时 CAPE 失效？

**解释**:
1. 高利率 → 估值自然受压
2. 收益率竞争 → 估值难以过度扩张
3. 利率本身成为主导因素
4. CAPE 信号被利率因素覆盖
